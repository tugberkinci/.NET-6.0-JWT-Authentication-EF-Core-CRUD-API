# JWT AUTHENTICATION

An CRUD rest API with JWT authentication.

## Features
- Get all data.
- Insert new data.
- Update exist data.
- Delete data from database.



## Tech
- .NET 6.0
- Entity Framework
- MySql
- JWT

## Installation
Update database connection settings which is in the DBContext file with your connection settings.
